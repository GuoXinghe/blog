快速解决

在 SQL Server 中，您可以通过以下步骤设置主键自增：
在创建表时，在定义主键的列时，使用 IDENTITY 关键字来指定该列为自增列。例如：
CREATE TABLE 表名 (
列名 INT PRIMARY KEY IDENTITY(1,1),
其他列名 数据类型,
…
);

如果表已经存在，您可以使用 ALTER TABLE 命令来修改现有列，使其成为自增列。例如：
ALTER TABLE 表名
ADD 列名 INT IDENTITY(1,1) PRIMARY KEY;
