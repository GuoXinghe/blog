MarkDown段落

我非常清楚这是一个段落。用换行隔开。

这是第二端文字。

这样也算是总分总结构了吧。

Markdown 换行语法

可以使用段落后加2个空格的方式进行换行。 也可使用“html语言的 br”
进行换行。

MarkDown 粗体斜体的使用

可以使用‘’单星号“斜体”*，双星号进行加粗，三星号 斜体与加粗

Markdown 引用语法

要创建块引用，请在段落前添加一个 > 符号。

这个就是引用语句，Hello MarkDown

大家都是一样的。

MarkDown列表语法

有序列表

要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点。数字不必按数学顺序排列，但是列表应当以数字 1 起始。

  1. 张三是一号
  2. 李四是二号

无序列表

要创建无序列表，请在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+) 。缩进一个或多个列表项可创建嵌套列表。

  • 无序王五
  • 无序赵六

Markdown 代码语法

要将单词或短语表示为代码，请将其包裹在反引号 () 中。 //导入java.utils包下的所有工具类<br>inmport java.utils.*`

Markdown 分隔线语法

要创建分隔线，请在单独一行上使用三个或多个星号 (***)、破折号 (—) 或下划线 (___) ，并且不能包含其他内容。

Markdown 链接语法

链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。

这是一个本人的Github地址(http://github.GuoXinghe.com)

Markdown 图片语法

要添加图像，请使用感叹号 (!), 然后在方括号增加替代文本，图片链接放在圆括号里，括号里的链接后可以增加一个可选的图片标题文本。

#Markdown 转义字符语法
要显示原本用于格式化 Markdown 文档的字符，请在字符前面添加反斜杠字符 \ 。

*加粗斜体被转义了*

Markdown 内嵌 HTML 标签

对于 Markdown 涵盖范围之外的标签，都可以直接在文件里面用 HTML 本身。如需使用 HTML，不需要额外标注这是 HTML 或是 Markdown，只需 HTML 标签添加到 Markdown 文本中即可。
